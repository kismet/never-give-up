# Scopo
I seguenti obiettivi anche se numerati possono essere realizzati in qualsiasi ordine. 
**Si ricorda** di effettuare un buon numero di commit come da best practice di git.

## Obiettivo 1 - Implementazione Libreria
Creare un implementazione delle funzioni definite nel file libs.h all'interno del file libs.cpp

## Obiettivo 2 - Test della Libreria
Apportare le necessarie modifiche al file main.cpp per utilizzare la libreria appena implementata, in particolare far in modo che nel main si possa provare tutte le funzioni e tutti i casi di test

## Obiettivo 3 - Autori
I tutti i file che andremo modificare deve essere inserito all'inizio del file come commento la seguente riga:
```
// Autore: (Cognom Nome)
// Data:   (aaaa/mm/gg)
// Classe: 3CI
// Fila:    (?????)
```
Sostituendo `cognome` con il proprio cognome.
